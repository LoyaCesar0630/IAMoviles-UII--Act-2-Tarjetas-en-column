"Actúa como un desarrollador experto en Flutter y Dart. Necesito crear una interfaz que muestre una columna de 5 tarjetas (Cards) de información, similar a un feed de proyectos o usuarios.

Requerimientos técnicos:

Modelo de Datos: Crea una clase llamada CardData que contenga imageUrl, username y title.

Diseño de la Tarjeta: Cada tarjeta debe tener un estilo horizontal (Row). A la izquierda, una imagen cuadrada de 120x120 con BoxFit.cover y un errorBuilder que muestre un icono de imagen rota si falla la carga. A la derecha, una columna con el username (texto normal) y el title (texto grande y en negrita).
